# Site - Sistema de Manutenção (início do zero)

Este pacote foi preparado para rodar em **GitHub Pages**.

## Banco (seed)
Os arquivos abaixo iniciam vazios (começando do zero):
- `data/equipes.json`
- `data/solicitacoes.json`

Na primeira execução, o sistema tenta carregar esses JSON.
Se não conseguir, ele inicia com listas vazias e grava no LocalStorage do navegador.

## Publicar
1. Faça upload de tudo deste ZIP no seu repositório
2. Ative GitHub Pages apontando para a branch/pasta raiz
3. Acesse o link do Pages

> Para resetar o “banco” local do navegador: limpe o LocalStorage do site ou use janela anônima.
